Clock output example

Example description
This example shows how to use the SYSCTL driver to drive a specific
clock source on the CLKOUT pin. To use this example, you'll need to
connect an oscilloscope to the CLKOUT pin on your board.

Special connection requirements
On LPC1769:
- PIO1_27 - CLKOUT (PAD10 on LPCXpresso board)
On LPC1788 and LPC4088:
- PIO1_27 - CLKOUT (J3-37 on EA OEM Base Board)
